Please write a function to reverse the following nested input value into output value

#### Input:

  'hired': {
    'be': {
      'to': {
        'deserve': 'I'
      }
    }
  }
}

#### Output:
output_value = {
  'I': {
    'deserve': {
      'to': {
         'be': 'hired'
      }
    }
  }
}
